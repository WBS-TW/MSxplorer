# MDPlotR
Using Shiny to analyze MS data by mass defect plots

MDPlotR is now part of the EnviGCMS package. 

---Instructions to install and run---

library(devtools)
<br>
devtools::install_github("yufree/enviGCMS")
<br>
library(enviGCMS)
<br>
enviGCMS::runMDPlot()
